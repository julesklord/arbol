# Soul & Identity

## The Project's Soul

A beautiful, tiny system dashboard for the terminal, focusing on speed, zero dependencies, and visual clarity.

## Guiding Principles

1. **Radical Simplicity**: If it's complex, it's not finished. Simple as that.
2. **Functional Aesthetics**: Beautiful code and beautiful CLI output is correct. Don't make it ugly.
3. **User Empathy**: We design for terminal enthusiasts who appreciate elegant layouts and immediate feedback.
